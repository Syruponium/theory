# Simulation Results & Mock Observables

This directory contains the high-resolution visual outputs from the Syruponium V8 calibration suite. These figures serve as the primary evidence for the model's predictive power and are used in the main project documentation.

### Contents:
* **fig1_lensing_residual_map.png**: Spatial correlation analysis between thermal flux ($\Phi$) and lensing residuals ($\kappa$).
* **fig2_centroid_relaxation_curve.png**: Temporal decay study showing the condensate healing timescale $\tau_s$.
* **fig3_chromatic_shift_correlation.png**: The primary falsification test showing wavelength-dependent centroid shifts.

All data points were generated using the scripts located in the `/scripts` directory.
