# Syruponium V8 Simulation Results

This directory contains the primary visual outputs from the V8 calibration suite:

1. **fig1_lensing_residual_map.png**: Maps the correlation between local heat flux and lensing anomalies.
2. **fig2_centroid_relaxation_curve.png**: Displays the temporal relaxation of lensing centroids post-collision.
3. **fig3_chromatic_shift_correlation.png**: Quantifies the predicted arcsecond shift across different wavelengths.

All plots were generated using the scripts found in the `/scripts` directory.
