# Syruponium V8 Simulation Results

This directory contains the primary visual outputs from the V8 calibration suite:

1. **Mock_residual.png**: Maps the correlation between local heat flux and lensing anomalies.
2. **Rotation_curve.png**: Displays the temporal relaxation of lensing centroids post-collision.
3. **Chromatic_shift.png**: Quantifies the predicted arcsecond shift across different wavelengths.

All plots were generated using the scripts found in the `/scripts` directory.
